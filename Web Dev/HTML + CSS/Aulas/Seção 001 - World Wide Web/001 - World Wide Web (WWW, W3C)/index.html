1 - World Wide Web (WWW, W3C)

A World Wide Web (Rede de Alcance Mundial) é uma rede de conteúdo online formatado em HTML e acessado via HTTP através de um browser.

De forma mais simples, é um sistema de documentos em hypermedia que são interligados e executados na internet, que acessamos com os navegadores.
