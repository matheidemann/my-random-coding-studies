4 - INTRODUÇÃO AO CSS

2 - Regras, seletores, declarações, propriedades e valores

CSS significa Cascading Style Sheets (Folha de Estilos em Cascata);

1) Regra (rule): consiste em um selector (seletor) e um declaration block (bloco de declaração), sendo que dentro deste ficarão as properties (propriedades) e os values (valores). Ela possui uma sintaxe própria que define como será aplicado estilo a um ou mais elementos da marcação HTML de uma página. Um conjunto de regras CSS formam uma folha de estilos.

1.1) Seletor (selector): é um elemento da marcação HTML identificado pelo nome da tag (<p>, <h1>, <form>, etc), classe (topo, .principal, .menu, etc), identificador (#tudo, #auxiliar, #rodape, etc), conjunto de tags, classes e identificadores, ou asterisco (* seleciona todo o documento HTML). O selector é dividido em cinco categorias:

1.1.1) Seletor simples (simple selector): seleciona elementos com base no nome da tag, classe ou id;

1.1.2) Seletor combinado (combined selector): seleciona os elementos com base em uma relação específica entre eles;

1.1.3) Seletor de pseudo-classe (pseudo-class selector): seleciona os elementos com base em uma determinada situação;

1.1.4) Seletor de pseudo-elementos (pseudo-elements selector): seleciona apenas parte de um elemento;

1.1.5) Seletor de atributo (attribute selector): seleciona elementos com base em um atributo ou valor de um atributo.

1.2) Bloco de declaração (declaration block): é o conjunto de uma ou mais propriedades e um valores, aqui é onde ocorre a estilização das marcações de HTML.

1.2.1) Propriedade (property): é a propriedade do elemento HTML ao qual será aplicada a estilização definida no valor;

1.2.2) Seletor de atributo (attribute selector): é a característica específica a ser assumida pela propriedade.

Regra de CSS
Seletor Declaração
h1 (ou qualquer outra forma de seleção) {font-size: 12;}
Propriedade Valor

Resultado:h1 {font-size:12;}



Regra de CSS com mais de um seletor e declração
Seletor Declaração Declaração
h1, .classe (ou qualquer outra forma de seleção) {font-size: 12; color: red;}
Propriedade Valor Propriedade Valor

Resultado:h1, .classe {font-size:12; color:red;}



OBS¹:é possível utilizar mais de uma declaration dentro de uma rule, assim possibilitando estilizar mais de uma propriedade de um selector. Para isso, basta colocar um ; após cada declaration, e então prosseguir escrevendo a próxima declaration. Exemplo: h1 {font-size:12; color:blue;}

OBS²:caso o valor da propriedade seja uma palavra composta, esta deverá estar entre aspas simples ou duplas, como no caso de definir o tipo de fonte Exemplo: h1 {font-family:”Comic Sans MS”;)

OBS³:é recomendável sempre colocar um ; após a última declaration, para no caso de futuramente ter que se acrescentar mais declarações na regra de CSS (evitando correr o risco de esquecer eventual ponto-e-vírgula inicialmente omitido).
