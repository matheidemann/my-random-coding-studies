5 - PROPRIEDADES BÁSICAS DE CSS

14 - Estilização de tabelas

Para personalizar tabelas podemos utilizar diversas propriedades a serem aplicadas tanto na tabela, quanto nas colunas ou nas linhas. Vale destacar que este tópico não tem apenas a finalidade de demonstrar como as listas podem ser personalizadas, mas sim como todos elementos de HTML podem ser estilizados com as mais diversas propriedades possíveis, isto é, a grande maioria das propriedades deste tópico se aplicam também, por exemplo, para as listas.

1) Adicionar umas borda da tabela: adicionar a propriedade “border”;

Exemplo de tabela com "border: 5px groove green;":

Coluna 1 (th) Coluna 2 (th) Coluna 3 (th)
Item 1 (td) Item 2 (td) Item 3 (td)
Item 1 (td) Item 2 (td) Item 3 (td)
Item 1 (td) Item 2 (td) Item 3 (td)

2) Remover as “bordas duplas”: utilizar a propriedade “border-collapse” com o valor “colapse”. Isto fara com que as linhas se colapsem, removendo o efeito de “bordas duplas”;

Exemplo de tabela com "border-collapse: collapse;":

Coluna 1 (th) Coluna 2 (th) Coluna 3 (th)
Item 1 (td) Item 2 (td) Item 3 (td)
Item 1 (td) Item 2 (td) Item 3 (td)
Item 1 (td) Item 2 (td) Item 3 (td)

Exemplo de tabela sem o "border-collapse: collapse;":

Coluna 1 (th) Coluna 2 (th) Coluna 3 (th)
Item 1 (td) Item 2 (td) Item 3 (td)
Item 1 (td) Item 2 (td) Item 3 (td)
Item 1 (td) Item 2 (td) Item 3 (td)

3) Modificar propriedades gerais da tabela, dos heads ou itens: utilizar as mais diversas propriedades no CSS para modificar cada elemento (<table>, <tr>, <th>, <td>);

Exemplo de tabela com várias propriedades alteradas:

Coluna 1 (th) Coluna 2 (th) Coluna 3 (th)
Item 1 (td) Item 2 (td) Item 3 (td)
Item 1 (td) Item 2 (td) Item 3 (td)
Item 1 (td) Item 2 (td) Item 3 (td)

4) Alinhar heads ou itens horizontalmente: utilizar a propriedade “text-align” e definir o valor desejado;

Exemplo de tabela com "text-align: right;" para os elementos <th> e "text-align: center;" para os elemntos <td>:

Coluna 1 (th) Coluna 2 (th) Coluna 3 (th)
Item 1 (td) Item 2 (td) Item 3 (td)
Item 1 (td) Item 2 (td) Item 3 (td)
Item 1 (td) Item 2 (td) Item 3 (td)

5) Alinhar heads ou itens verticalmente: utilizar a propriedade “vertical-align” e definir o valor desejado (a tabela deve conter espaço suficiente para o item poder realmente se mover);

Exemplo de tabela com "vertical-align: bottom;" nos elementos <td>:

Coluna 1 (th) Coluna 2 (th) Coluna 3 (th)
Item 1 (td) Item 2 (td) Item 3 (td)
Item 1 (td) Item 2 (td) Item 3 (td)
Item 1 (td) Item 2 (td) Item 3 (td)

6) Alinhar a tabela ao centro: utilizar a propriedade “margin” com os valores “0px auto” para a tag “table”;

Exemplo de tabela com "margin: 0px auto;":

Coluna 1 (th) Coluna 2 (th) Coluna 3 (th)
Item 1 (td) Item 2 (td) Item 3 (td)
Item 1 (td) Item 2 (td) Item 3 (td)
Item 1 (td) Item 2 (td) Item 3 (td)

7) Aumentar o tamanho de cada linha: utilizar a propriedade “padding” com o valor desejado;

Exemplo de tabela com "padding: 30px;" nos elementos <th>:

Coluna 1 (th) Coluna 2 (th) Coluna 3 (th)
Item 1 (td) Item 2 (td) Item 3 (td)
Item 1 (td) Item 2 (td) Item 3 (td)
Item 1 (td) Item 2 (td) Item 3 (td)

8) Colocar uma borda apenas na parte inferior de cada linha: utilizar a propriedade “border-bottom” para cada head e item;

Exemplo de tabela com "border-bottom”: 2px;":

Coluna 1 (th) Coluna 2 (th) Coluna 3 (th)
Item 1 (td) Item 2 (td) Item 3 (td)
Item 1 (td) Item 2 (td) Item 3 (td)
Item 1 (td) Item 2 (td) Item 3 (td)

9) Mudar a certas propriedades visuais de cada linha ao passar o mouse por cima: utilizar o seletor “:hover” no , e então definir as declarações desejadas;

Exemplo de tabela com "hover” nos elementos <td> (passe o mouse por cima deles):

Coluna 1 (th) Coluna 2 (th) Coluna 3 (th)
Item 1 (td) Item 2 (td) Item 3 (td)
Item 1 (td) Item 2 (td) Item 3 (td)
Item 1 (td) Item 2 (td) Item 3 (td)
