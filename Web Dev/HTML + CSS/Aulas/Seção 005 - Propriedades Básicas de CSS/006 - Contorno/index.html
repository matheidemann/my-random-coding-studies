5 - PROPRIEDADES BÁSICAS DE CSS

6 - Contorno

Um outline (contorno) é uma linha que é desenhada fora da border do elemento, ou seja, é possível utilizar a outline junto de uma border.

A grande diferença de uma outline para uma border é que a outline é desenhada fora do elemento e pode sobrepor outros conteúdos, e ainda, a outline não faz parte das dimensões do elemento, isto é, a altura e largura do elemento não são afetadas por ela.

As outlines possuem diversas propriedades, sendo elas:

1) outline-style: define o estilo do contorno (possui os mesmos estilos da border)

Exemplo:

Elemento de parágrafo com outline-style como "dashed"


2) outline-color: define a cor do contorno;

Exemplo:

Elemento de parágrafo com uma outline de cor azul.


3) outline-width: define a grossura do contorno;

Exemplo:

Elemento de parágrafo com uma outline de 5px.


4) outline-offset: define uma distância entre o contorno e a borda do elemento;

Exemplo:

Elemento de parágrafo com outline-offset de 5px.


5) outline: é a shorthand property para o contorno.

Exemplo:

Elemento de parágrafo com uma outline de 15px, double e de cor amarela.


OBS¹: definir apenas a grossura e cor do contorno não funciona, isto é, precisa ser definido pelo menos o estilo do contorno para que ela seja visível.
