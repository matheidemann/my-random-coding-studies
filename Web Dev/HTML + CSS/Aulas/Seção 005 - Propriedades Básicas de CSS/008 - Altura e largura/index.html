5 - PROPRIEDADES BÁSICAS DE CSS

8 - Altura e largura

Assim como é possível definir a altura e largura de imagens, é possível definir estas propriedades para qualquer elemento. Como padrão, podemos utilizar qualquer tipo de medida para definir a altura e largura de um elemento.

1) width: define a largura do elemento;

2) height: define a altura do elemento.

Porém, caso usemos estas propriedades, uma vez que redimensonamos nosso navegador, o elemento continuará possuindo 1280x720px, e isso pode ser um problema para telas menores, ou mesmo para dispositivos mobile (tente redimensionar seu navegador e veja que a imagem não ficará do tamanho do navegador).

Exemplo:


Para resolver isso, há quatro propriedades essenciais para auxiliar com o design responsivo (página que responde corretamente para qualquer tamanho de tela):

1) max-width: define a largura máxima do elemento;

2) max-height: define a altura máxima do elemento.

3) min-width: define a largura mínima do elemento;

4) min-height: define a altura mínima do elemento.


Exemplo:

Imagem de 1920x1080px com max width e height de 480x270px.

Assim, caso fosse redimensionado o navegador, ainda assim o elemento cabeira normalmente e funcionalmente na tela, sem o problema de parte dele ficar escondido na lateral, fazendo com que o usuário seja obrigado a utilizar a barra de scroll inferior para poder vê-lo novamente.


Exemplo:

Imagem de 480x270px com min width e height de 1280x720px.

Ainda, a ideia com o “min” é a mesma, porém de forma invertida, ou seja, se o meu documento tem a largura mínima de 500px, quando eu abrir a página em uma tela de 1920px de largura, o elemento irá ocupar os 1920px da tela, e na medida que ele for sendo redimensionado, quando chegar em 500px, ele irá parar de diminuir de largura e ficar fixo em 500px, criando uma barra de scroll inferior dos 499px para baixo.


OBS¹: o valor da propriedade “max-width” sobrescreve o valor da propriedade “width”, e o mesmo se aplica ao “max-height” e “height”.
