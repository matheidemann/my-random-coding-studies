5 - PROPRIEDADES BÁSICAS DE CSS

5 - Espaçamento Interno

O espaçamento interno no CSS é chamado de padding, e com ele criamos espaço entre o conteúdo do elemento, dentro de quaisquer bordas definidas.

1) padding-top: define a margem superior para o elemento.

Exemplo:

Elemento de parágrafo com padding-top de 50px


2) padding-right: define a margem da direita para o elemento.

Exemplo:

Elemento de parágrafo com padding-right de 50px (visualmente não ocorre nada pois o elemento parágrafo está alinhado à esquerda.)


3) padding-bottom: define a margem inferior para o elemento.

Exemplo:

Elemento de parágrafo com padding-bottom de 50px


4) padding-left: define a margem da esquerda para o elemento.

Exemplo:

Elemento de parágrafo com padding-left de 50px


5) padding: é a shorthand property para o padding.

Exemplo:

Elemento de parágrafo com padding de 50px


OBS¹:o padding adiciona mais pixels para os elementos. Isto é. Se eu defini que meu elemento tem 300px de largura, quando eu aplicar um padding de 50px em cada lado, logo ele passará a ter 400px no total, uma vez que o padding não colapse com o tamanho já pré-definido do elemento. Caso não seja a ideia aumentar o tamanho do elemento, podemos usar a declaração “box-sizing: border-box;”. Está propriedade será estudada na sua respectiva seção.
