5 - PROPRIEDADES BÁSICAS DE CSS

10 - Fontes de texto

No CSS há cinco tipos diferentes de fontes genéricas:

1) Serif: são fontes que possuem pequenos contornos nas bordas de cada letra, trazendo um visual mais elegante e formal.

Exemplo:

Times New Roman


2) Sans-Serif: são fontes limpas, que não possuem contornos em suas bordas, trazendo um visual mais simples e minimalista.

Exemplo:

Verdana


3) Monoscape: são fontes que todas as letras possuem exatamente a mesma largura, trazendo um visual mais mecânico.

Exemplo:

Consolas


4) Cursive: são fontes que imitam a escrita à mão.

Exemplo:

Comic Sans MS


5) Fantasy: são fontes que trazem visuais mais decorativos.

Exemplo:

Impact


Podemos utilizar qualquer fonte para estilizar nossa página, e para isso o CSS fornece diversas propriedades:

1) font-family: define a fonte que será usada. Aqui deve ser introduzido o nome da fonte;

Exemplo:

Elemento de parágrafo com "font-family: 'Courier';


2) font-size: define o tamanho da fonte, aceitando todos os tipos de medidas;

Exemplo:

Elemento de parágrafo com "font-size 24px"


3) font-style: define se a fonte será normal, itálica ou oblíqua (é a mesma coisa que itálico);

Exemplo:

Elemento de parágrafo com "font-style: italic;"


4) font-variant: define se a fonte será normal ou então será em “small-caps” (todas as letras permanecem do mesmo tamanho, porém ficam em CAPS LOCK);

Exemplo:

Elemento de parágrafo com "font-variant: small-caps"


5) font-weight: define se a fonte será normal ou em bold (negrito). Essa propriedade aceita outras valores, mas dificilmente são utilizados, como “bolder”, “lighter” e até valores de 100 a 900, mudando apenas a centena;

Exemplo:

Elemento de parágrafo com "font-weight: bold"


6) font: é a shorthand property para modificar a fonte, para especificar todas as propriedades individualmente em uma única linha.

Exemplo:

Elemento de parágrafo com "font: italic small-caps bold 22px 'Arial Black';"


Por fim, é válido destacar que é possível importar fontes customizadas para dentro da page. Para isso, é necessário definir no CSS um comando que criará uma nova fonte:

@font-face {font-family:'Nome da fonte'; src:url(caminho/fonte.ttf)};

Após isto, basta utilizar a fonte normalemente com o nome inserido no "font-family".

Exemplo:

Elemento de parágrafo com "font-family: base-02.ttf";"


OBS¹: as melhores fontes para serem utilizadas em uma page são: Arial, Verdana, Helvetica, Tahoma, Trebuchet MS, Times New Roman, Georgia, Garamond, Courier New, Brush Script MT.
