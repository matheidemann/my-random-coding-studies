15 - Colunas

2) column-rule: shorthand property para definir as propridades da linha que separa as colunas;

2.1) column-rule-width: define a grossura da linha;

2.2) column-rule-style: define o estilo da linha (aceita os valores de estilo para bordas, contornos etc);

2.3) column-rule-color: define a cor da linha;
