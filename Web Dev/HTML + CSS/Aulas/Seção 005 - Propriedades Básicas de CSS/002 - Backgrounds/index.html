5 - PROPRIEDADES BÁSICAS DE CSS

2 - Backgrounds

Há várias formas para definir uma cor no CSS. Diversos elementos podem ter suas cores editadas, como texto, background, bordas etc. Para modificar as cores podemos utilizar valores em:

1) background-color: é utilizada para definir a cor de fundo de um elemento.

Exemplo:

Elemento de parágrafo com a cor azul clara de fundo


2) background-image: é utilizada para definir uma imagem como fundo de algum elemento. É necessário um atributo “url” para especificar qual será a imagem que ficará de fundo. Ainda, por padrão, a imagem é repetida no eixo horizontal e vertical, isto é, quando a imagem acaba, ela começa de novo no fundo.

Exemplo:

Elemento de parágrafo com imagem de fundo (tamanho de 480x270px).


3) background-repeat: é utilizada para definir se a imagem irá se repetir ou não. É possível utilizar os valores:

3.1) repeat-x: a imagem será repetida apenas na horizontal;

Exemplo:

Elemento de parágrafo com imagem sendo repetida apenas na horizontal.


3.2) repeat-y: a imagem será repetida apenas na vertical;

Exemplo:

Elemento de parágrafo com imagem sendo repetida apenas na vertical.


3.3) no-repeat: a imagem não será repetida;

Exemplo:

Elemento de parágrafo com imagem não sendo repetida.


4) background-position: altera a posição padrão do background. É possível utilizar diversos valores de medidas aqui, desde os padrões (top, bottom, top left, bottom right, center etc) até os mais complexos (cm, em, ch, %, etc).

Exemplo:

Elemento de parágrafo com imagem na posição 'top right"


5) background-attachment: é utilizada para definir se a imagem ficará fixa ou não quando se rola o scroll do mouse.

Exemplo:

Olhe para o canto superior direito da tela. O valor definido foi "fixed, pois o valor "scroll" já é o valor padrão.


6) background: é a shorthand property para as configurações de background.

Exemplo:

Elemento de parágrafo com imagem sendo estilizada por diversas propriedades em apenas uma linha + opacity.



OBS¹: é possível alterar a cor de fundo de qualquer elemento, seja um parágrafo (>p>), um título (>h1>) ou toda a página (<body>) etc.

OBS²: é possível alterar a opacidade de elementos com a declaração “opacity”;

OBS³: na questão da posição do background, é possível colocar a horizontal ou vertical em qualquer sequência, isto é, tanto "top right" quanto "right top" trarão o mesmo resultado.
