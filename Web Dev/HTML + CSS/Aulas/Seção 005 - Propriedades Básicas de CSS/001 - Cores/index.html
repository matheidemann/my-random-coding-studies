5 - PROPRIEDADES BÁSICAS DE CSS

1 - Cores

Há várias formas para definir uma cor no CSS. Diversos elementos podem ter suas cores editadas, como texto, background, bordas etc. Para modificar as cores podemos utilizar valores em:

1) RGB: rgb(50,0,0)

2) RGBA: rgba(50,0,0)

3) HEX: #d261ff

4) HSL: hsl(200,100%,50%)

5) HSLA: hsla(66,100%,69%,0.55)

6) Nome: lightgreen


Além de utilizar cores simples, é possível utilizar gradientes lineares ou radiais:

1) Gradiente linear: faz com que o elemento tenha uma cor gradiente de forma linear, isto é, vai para baixo, direita, esquerda, para cima ou na diagonal:

Exemplos:

background: linear-gradient(red, yellow);

background: linear-gradient(to right, green, blue);

background: linear-gradient(to bottom left, green, blue);

background: linear-gradient(red, yellow, blue, purple);

background: background: linear-gradient(70deg, rgb(255,153,153), rgb(0,153,0));


2) Gradiente radial: faz com que o elemento tenha uma cor gradiente de forma radial, isto é, vai do centro para fora:

Exemplos:

background: radial-gradient(red, yellow);

background: radial-gradient(green, blue);

background: radial-gradient(red, yellow, blue, purple);

background: radial-gradient(rgb(255,153,153), rgb(0,153,0));


OBS¹: não há uma declaration específica para o texto, como por exemplo “font-color”. O correto, para modificar a cor do texto, é apenas escrever “color”. Porém, outros elementos possuem certas declarations específicas, como “background-color” e “border-color”.
