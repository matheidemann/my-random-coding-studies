5 - PROPRIEDADES BÁSICAS DE CSS

9 - Formatação de texto

Sobre a formatação do texto no CSS, podemos definir diversas propriedades, sendo elas relacionadas a cor, alinhamento, decoração, transformação, espaçamento e fonte.

1) Propriedades de cor:

1.1) color: utilizada para mudar a cor do texto, compreende quaisquer valores de cor;

Exemplo:

Elemento de parágrafo com "color: red;"



1.2) background-color:

Exemplo:

Elemento de parágrafo com "background-color: lightgrey;"



2) Propriedades de alinhamento:

2.1) text-align: utilizada para definir o alinhamento horizontal do texto segundo onde ele está presente. Compreende os valores “middle”, “left”, “right” e “justify” (justificar texto);

Exemplo:

Elemento de parágrafo com "text-align: right;".


2.2) vertical-align: utilizada para definir o alinhamento vertical do texto segundo onde ele está presente. Compreende os valores “top”, “middle”, e “bottom”;

Exemplo:

Elemento de parágrafo com uma imagem em "vertical-align: middle;"


2.3) direction e unicode-bidi: utilizada para colocar o texto sendo escrito da direita para a esquerda e com as letras invertidas. É provavelmente utilizado para escritas que precisem começar da direita, como o Árabe. Para usar essas propriedades, o direction deve ter o valor “rtl” e o Unicode-bidi o valor “bidi-override”.

Exemplo:

Elemento de parágrafo com "direction: rtl;" e "unicode-bidi: bidi-override;"


3) Propriedades de decoração:

3.1) text-decoration: utilizada para colocar decorações no texto. Compreende os valores “none”,“overline”, “line-through” e “underline”;

Exemplo:

Elemento de parágrafo com "text-decoration: overline;"


3.2) text-shadow: utilizada para adicionar uma sombra atrás do texto. Para definir seus valores, é necessário colocar “text-shadow: tamanho tamanho desfoque cor”.

Exemplo:

Elemento de parágrafo com "text-shadow: 1px 1px red;"


4) Propriedades de transformação:

4.1) text-transform: utilizada para transformar as letras do texto, podendo deixa-las em maiúsculas, minúsculas ou de forma capitalizada (cada palavra começa com uma letra maiúscula). Compreende os valores “uppercase”, “lowercase” e “capitalize”.

Exemplo:

Elemento de parágrafo com "text-transform: capitalize;"


5) Propriedades de espaçamento:

5.1) text-indent: utilizada para definir um recuo no início do texto. Compreende valores de tamanho;

Exemplo:

Elemento de parágrafo com "text-indent: 15px"


5.2) letter-spacing: utilizada para definir o espaço entre as letras. Compreende valores de tamanho;

Exemplo:

Elemento de parágrafo com "letter-spacing: 5px;"


5.3) line-height: utilizada para definir o espaço entre as linhas. Compreende valores de tamanho;

Exemplo:

5.4) word-spacing: utilizada para definir o espaço entre as linhas. Compreende valores de tamanho;

Exemplo:

Elemento de parágrafo com "word-spacing: 15px;"


5.5) white-space: utilizada para desabilitar a quebra automática de texto, assim como desrespeitar os limites de tamanho horitontal do elemento inserido (nowrap), para modificar um elemento e permitir texto pré-formatado (pre) ou então para permitir texto pré-formatado sem respeitar os múltiplos espaços (pre-line).
