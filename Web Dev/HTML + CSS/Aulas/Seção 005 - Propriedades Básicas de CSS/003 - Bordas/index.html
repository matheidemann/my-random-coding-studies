5 - PROPRIEDADES BÁSICAS DE CSS

3 - Bordas

É possível modificar diversos aspectos da propriedade “border”, como estilo, tamanho e cor.

1) border-width: é utilizada para definir a grossura da borda. Esta propriedade aceita diversas medidas diferentes (px, pt, cm, em, etc), e também possui algumas padrões como thin (fina), medium (média) e thick (grossa). A grossura padrão é de 1px.

Exemplo:

Elemento de parágrafo com uma borda de 15px.


2) border-style: é utilizada para definir o estilo da borda. Há diversos estilos de borda, sendo eles: dotted, dashed, solid, double, groove, ridge, inset, outset, none e hidden. O estilo padrão é o solid.

Exemplo:

Elemento de parágrafo com uma borda no estilo "dotted".


3) border-color: é utilizada para definir a cor da borda. Todos os valores de cor são aceitos aqui (rgb, rgba, hex, hsl, hsla e name). A cor padrão da borda é preto.

Exemplo:

Elemento de parágrafo com uma borda de cor vermelha.


4) border: é a shorthand property para as configurações de borda.

Exemplo:

Elemento de parágrafo com uma borda de 20px, verde e no estilo "groove".


5) border-radius: é utilizada para deixar as bordas mais redondas. É algo que não vem por padrão, e também não pode ser colocada na mesma linha dos valores da shorthand property “border”.

Exemplo:

Elemento de parágrafo com o raio da borda em 25px.


6) border-image: é utilizada para definir o estilo da borda com uma imagem.

Exemplo:

Elemento de parágrafo com borda de imagem em forma "round" com 30 de slice.

Elemento de parágrafo com borda de imagem em forma "round" com 15 de slice.

Elemento de parágrafo com borda de imagem em forma "stretch" com 30 de slice.

Elemento de parágrafo com borda de imagem em forma "stretch" com 15 de slice.


OBS¹: definir apenas a grossura e cor da borda não funciona, isto é, precisa ser definido pelo menos o estilo da borda para que ela seja visível;

OBS²: é possível definir apenas um lado da borda para ser arredondado, e para isso basta colocar a propriedade e os valores "border-radius: superior-esquerdo superior-direito inferior-direito inferior-esquerdo";

OBS³: é possível colocar uma barra "/" para dois valores do border-radius, podendo criar bordas elípticas, por exemplo "border-radius: 50px / 15px;"
