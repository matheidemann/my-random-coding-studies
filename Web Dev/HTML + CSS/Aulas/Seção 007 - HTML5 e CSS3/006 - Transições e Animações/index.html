6 - Transições e Animações

Como o próprio nome sugeste, as transições são utilizadas para "transicionar" os elementos de uma estilização para outra, com um efeito suave, podendo variar sua duração, atraso, maneira de transição etc.

1) transition: é a shorthand property para o uso das transições. No seu valor deve ser colocado a propriedade a ser alterada, a duração para ocorrer a transição total, a forma de transição e o atraso (transition:width 2s ease 1s, height 3s linear 0.5s;).

2) animation: é a shorthand property para o uso das animações. No seu valor deve ser colocado o nome da animação, a duração da animação, a forma de transição, o atraso, a quantidade de vezes que a animação será tocada e a direção da animação (animation: animacao1 4s linear 1s infinite alternate;).

As animações sáo extremamente semelhantes as transições, sendo sua maior diferença o fato de ser uma propriedade mais poderosa, podendo animar cada keyframe da animação (com o seletor de keyframe da animação).

E por fim, vale destacar que para mover os elementos com as transições/animações é necessário que o elemento seja posicionado (posição diferente de static).

2.1) animation-name: é o nome da animação, sendo que este nome será usado para utilizar o seletor de keyframes (para especificar qual animação deve ser estilizada).
