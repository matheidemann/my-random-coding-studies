1 - Metadados e SEO

Para que o site fique mais completo e seja melhor encontrado nas search engines é utilizado os metados em HTML. Os metadados são colocados dentro da tag head, utilizando a tag meta.

1) name="description": metatag utilizada para definir a descrição do site;

2) name="keywords": metatag utilizada para definir as palavras-chave do site (se meu site for de animais eu devo utilizar keywords que tenham a ver com animais, para caso alguém pesquisar sobre animais em uma search engine encontrar meu site);

3) name="author": metatag utilizada para definir o autor do site ou nome da empresa;

4) name="viewport": metatag utilizada para ajustar a página para o responsivo para dispositivos mobile;


Fora as metataga específicas listadas acima, também é importante definir o nome de imagens e da navbar utilizando o atributo "title".

Exemplos:

<a title="Home" href="www.teste.com">Home</a>

<a title="Sobre" href="www.teste.com">Sobre</a>

<a title="Contatos" href="www.teste.com">Contatos</a>
