5 - Vídeo e áudio

Para adicionar vídeo a uma page de HTML basta utilizar a tag <video>.

Vale destacar que o vídeo deve estar em formato .mp4, e o vídeo sempre terá a mesma proporção, independetemente do tamanho que seja definido no CSS.
