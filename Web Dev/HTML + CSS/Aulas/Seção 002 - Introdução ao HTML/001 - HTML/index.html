2 - INTRODUÇÃO AO HTML

1 - HTML

HTML significa Hyper Text Markup Language (Linguagem de Marcação de Hipertexto);

É uma linguagem de marcação (não é uma linguagem de programação);

É utilizada para definir a estrutura de páginas da web;

Consiste em uma série de elements que dizem para o navegador como mostrar o conteúdo (título, corpo do texto, parágrafo, linha sem quebra de texto, a fonte da letra, as cores etc).
