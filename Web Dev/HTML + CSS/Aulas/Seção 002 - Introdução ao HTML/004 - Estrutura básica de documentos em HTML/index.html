4 - Estrutura básica de documentos em HTML

1) <!DOCTYPE html>: é a declaração que define que o documento é um documento em HTML5. Todos os documentos em HTML devem ser declarados com esta tag antes de qualquer tag. Ainda, todo documento de HTML não é case-sensitive.

2) <html>: é o elemento raiz de uma página em HTML.

2.1) lang: é um atributo para definir qual o idioma da webpage, sua utilidade se dá principalmente a respeito de search engines (mecanismos de pesquisa).

3) <head>: é o elemento que contém a metainformação da página em HTML.

3.1) <meta>: é comumente utilizado para definir o conjunto de caracteres da página, que no caso para todos os caracteres do português brasileiro funcionar é necessário o atributo "charset" com o valor "utf-8"

4) <title>: é o elemento que define o título da página em HTML.

5) <body>: é o elemento que define todo o corpo do documento, é o container que abrigará todo o conteúdo visual da página.


Exemplo:

<!DOCTYPE html> <html lang=”pt-BR”> <head> <meta charset="utf-8"> <title>Título</title> </head> <body> TODO A ESTRUTURA DA PAGE FICA AQUI </body> </html>

OBS¹: case-sensitive significa “sensível a maiúsculas e minúsculas”. Ou seja, aquilo que não é case-sensitive não precisa diferenciar maiúsculas ou minúsculas para funcionar corretamente.

OBS²: mesmo que o HTML não seja case-sensitive, é recomendável pela própria W3C que as tags e atributos sejam escritas em lowercase (caixa baixa), para assim funcionar em documentos mais restritos como os de XHTML.
