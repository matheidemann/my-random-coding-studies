6 - PROPRIEDADES MAIS COMPLEXAS DE CSS

6 - Alinhamento de texto

Há diversas formas para se alinhar texto no CSS. Para tal, serão utilizadas diversas propriedades que já foram vistas até agora. Vale destacar que certas propriedades também servem para outros elementos.

1) Alinhando bloco de texto no meio com margens: utilize a propriedade "margin: 0px auto" no elemento que ficará alinhado ao centro (preferencialmente uma div). Isso acontece pois a altura das laterais são dividas igualmente com o valor "auto".

Exemplo:

Elemento alinhado ao centro.

2) Alinhando texto com o "text-align": utilize a propriedade "text-align" e defina o valor desejado ("left", "right" ou "center")

Exemplo:

Texto alinhado ao centro do elemento.

3) Alinhando texto verticalmente com o "padding": utilize o padding para definir um valor para todos os lados igualmente. Detalhe que esta forma de alinha o texto verticalmente ao centro não funciona caso o elemento possua uma altura definida.

Exemplo:

Texto alinhado no meio do elemento com padding.

4) Alinhando texto verticalmente com o "line-height": utilize o line-height para definir um valor qualquer, sendo que, quanto maior este valor, maior será a altura do elemento. Detalhe que para calcular a altura total do elemento é necessário: calcular o tamanho da altura da linha + (o tamanho da fonte * a quantidade de linhas).
