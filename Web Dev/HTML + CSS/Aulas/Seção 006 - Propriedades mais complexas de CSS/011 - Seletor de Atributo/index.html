6 - PROPRIEDADES MAIS COMPLEXAS DE CSS

11 - Seletor de Atributos

Os seletores de atributos são seletores que selecionam certos elementos segundo seus atributos. Para utilizar um seletor de atributo, se usa os colchetes [attribute]. Há diversos tipos de seletores de atributo:

1) [attribute="value"] estiliza elementos que possuam determinado atributo específico com valor específico.

A string do valor a ser procurada deve ser especificamente igual e completa.

Exemplo (h2[class="title-middle-2"]):

title-start-1


title start 2


title middle 2


title-middle-2


endtext-1


2) [attribute~="value"] estiliza elementos que possuam uma string completa dentro de algum atributo.

A string do valor a ser procurada deve ser especificamente igual.

A string do valor a ser procurada pode estar em qualquer lugar.

A string do valor a ser procurada não será considerada se estiver escrita colada com outros caracteres (inclusive hífens);

Exemplo (a[class|="middle"]):

middle-1
middle 2
title middle 2
text-2
endtext-1

3) [attribute|="value"] estiliza elementos que comecem com uma string com um valor específico.

A string do valor a ser procurada deve ser especificamente igual.

A string do valor a ser procurada deve estar no início.

A string do valor a ser procurada não será considerada se estiver escrita colada com outros caracteres ou se estiver com espaços, porém, funcionará caso esteja colada com hífens;

Exemplo (p[class|="title"]):

title-start-1

text-title-1

title middle 1

text title 2

endtext-1


4) [attribute^="value"] estiliza elementos que comecem com uma string ou substring com um valor específico.

A string do valor a ser procurada não precisa ser especificamente igual (pode ser uma substring).

A substring do valor a ser procurada deve estar no início.

A substring do valor a ser procurada será considerada se estiver escrita colada com outros caracteres, se estiver com espaços ou hífens;

Exemplo (span[class|="t"]):

title-start-1
t-1
t middle-2
text t
end-t

5) [attribute$="value"] estiliza elementos que terminem com uma string ou substring com um valor específico.

A string do valor a ser procurada não precisa ser especificamente igual (pode ser uma substring).

A substring do valor a ser procurada deve estar no final.

A substring do valor a ser procurada será considerada se estiver escrita colada com outros caracteres, se estiver com espaços ou hífens;

Exemplo (pre[class|="ext"]):

title-start-text
text-1-ext
title middle ext
text-2
ext-endtext-1

6) [attribute*="value"] estiliza elementos que possuam uma string ou substring específica em qualquer lugar.

A string do valor a ser procurada não precisa ser especificamente igual (pode ser uma substring).

A substring do valor a ser procurada pode estar em qualquer lugar (início, meio, fim ou mesmo junta de outros caracteres).

A substring do valor a ser procurada será considerada se estiver escrita colada com outros caracteres, se estiver com espaços ou hífens;

Exemplo (h3[class*="tex"]):

title-start-1-ex

middle-text-1

title-middletext-2

my text 2

end tex-1
