14 - Media Queries

As Media Queries são utilizadas para redimensionar, adaptar e reorganizar o conteúdo da page segundo a largura e altura da viewport ou dispostivo, a resolução e/ou a orientação (se a tela está em modo paisagem ou retrato). Ainda, são de extrema importância para criar pages com design reponsivo, podendo se adaptar nos mais diversos tamanhos de tela.

As formas mais simples de como se utilizar as media queries são:

@media screen and (max-width/height: valor): define que, quando a tela for menor que determinado valor, a page receberá certas propriedades e valores;

@media screen and (min-width/height: valor): define que, quando a tela for maior que determinado valor, a page receberá certas propriedades e valores;

@media screen and (min-width/height: valor) and (max-width/height: valor): define que, quando a tela estiver entre o primeiro valor e o segundo, a page receberá certas propriedades e valores;
