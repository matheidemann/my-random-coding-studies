6 - PROPRIEDADES MAIS COMPLEXAS DE CSS

5 - Box Sizing

A propriedade box-sizing permite que seja tanto o espaçamento interno quanto a borda seja incluída sem desrespeitar o tamanho definido no elemento.

A largura e altura dos elementos são calculadas da seguinte forma por padrão: width + height + padding + border = tamanho total.

Ou seja, por padrão, o padding e a border serão somadas com o tamanho total do elemento, e portanto, de nada adianta querer um elemento de 100x100px, caso seu padding seja de 50px, pois o elemento passará a ter 150x150px. O nome deste tipo de box-sizing é "content-box", o valor padrão para todos os elementos.

Agora, com o valor "border-box" a largura e altura dos elementos serão calculadas da seguinte forma: width + height = tamanho total (não irã somar o tamanho do padding e da border).
