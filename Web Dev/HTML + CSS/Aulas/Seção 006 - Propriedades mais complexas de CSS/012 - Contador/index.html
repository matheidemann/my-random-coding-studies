6 - PROPRIEDADES MAIS COMPLEXAS DE CSS

12 - Contador

Contadores de CSS são variáveis que podem ter seu valor aumentado através de regras de CSS. Os contadores ajudam a numerar o conteúdo da página, sem a necessidade de mudar o número um por um.

Para realizar essa numeração de forma automática são utilizadas as seguintes propriedades:

1) counter-reset cria ou reseta um contador.;

2) counter-increment adicione um valor ao contador;

3) counter() ou counter() são funções com o objetivo de adicionar o valor do contator em um elemento;

4) content não é exclusivo do contador, mas é necessário para inserir o conteúdo gerado com o contador;

Os contadores também podem ser usados facilmente em listas. Para listas numeradas, o correto e prático é utilizar a função "counters()".
