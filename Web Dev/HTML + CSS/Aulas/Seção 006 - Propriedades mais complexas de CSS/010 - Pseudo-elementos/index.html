6 - PROPRIEDADES MAIS COMPLEXAS DE CSS

10 - Pseudo-elementos

As pseudo-elementos são utilizados para estilizar uma parte específica de um elemento, como a primeira linha, a primeira letra, colocar algum conteúdo antes ou após o elemento etc.

1) ::first-line: estiliza a primeira linha de um elemento. Detalhe que este pseudo-elemento pode apenas ser aplicado em elementos que possuem seu display em bloco;

Exemplo (.p1::first-line):

2) ::first-letter: estiliza a primeira letra de um elemento;

Exemplo (.p2::first-letter):

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Morbi vel magna nec dui ultrices placerat eget euismod urna. Lorem ipsum dolor sit amet, consectetur adipiscing elit. 3) ::before: insere algum conteúdo antes de algum determinado elemento;

4) ::after: insere algum conteúdo após algum determinado elemento;

Exemplo (.p4::after):

5) ::marker: estiliza apenas os marcadores de determinados elementos (com as bolinhas em uma lista não-ordenada);

Exemplo (.p5::marker):

  • Item 1
  • Item 2
  • Item 3

OBS¹: perceba que, diferentemente das pseudo-classe, os pseudo-elementos utilizam dois pontos "::". Isso é algo que foi introduzido no CSS3, enquanto nas versões anteriores (CSS1 e CSS2) ambos utilizavam apenas um dois pontos ":".
