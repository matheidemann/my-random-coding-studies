9 - Pseudo-classes - Parte 2

3) first-of-type: seleciona os primeiros elementos child de um tipo particular de cada parent.

No caso da utilização de classes, ele selecionára os primeiros elementos "únicos" de cada div;

5) nth-child: seleciona o elemento child específico de uma parent, com base em um número, palavra-chave ou fórmula.

Ou seja, caso o valor seja 2 para os elementos <p>, e eu possua um parágrafo na segunda posição de duas divs diferentes, esta pseudo-classe selecionará ambos os parágrafos.

No caso se utilizar classes, será a mesma coisa, porém com o valor 2 as segundas classes devem ser a específicada a ser selecionada (se eu quero selecionar text7 na posição 2, caso minha div tenha uma text8 na posição 2, ela não será selecionada);

6) nth-last-child: seleciona o elemento child específico de uma parent, com base em um número, palavra-chave ou fórmula, porém, isso é feito de trás para frente.

7) nth-of-type: seleciona os elementos child de tipos diferentes de cada parent, com base em um número, palavra-chave ou fórmula.

Ou seja, caso o nth-of-type com valor 1, e eu possua 3 divs, sendo cada uma delas composta de elementos diferentes, o seletor selecionará todos os 3 elementos, pois ele seleciona o primeiro elemento "único" (valor 1) de cada div.

E ainda, para ficar melhor exemplificado, caso meu nth-of-type seja de valor 2, com o mesmo exemplo acima, nenhum elemento será selecionado, pois esta pseudo-classe faz a conta com base nos elementos "únicos", e se todos 3 elementos dentro das divs são diferentes, logo eu nunca terei o 2° parágrafo, ou span, ou link dentro de uma div;

9) only-of-type: seleciona os elementos child únicos e não repetidos de cada parent.

Ou seja, utilizando classes, utilizando esta pseudo-classe, ela selecionará todos os elementos dentro de uma div que não possuírem outro igual, isto é, se houver um elemento de parágrafo dentro de uma div, ele será selecionado, mas caso haja dois, logo ele não será mais selecionado, pois passousse a repetir-se.

(Text13) 1° Parágrafo dentro da div1
