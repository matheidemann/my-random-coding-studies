13 - Object-fit e Object-position

O Object-fit define como uma imagem deve ser redimensionada para caber dentrode seu contêiner. Essas duas propriedades são extremamente importantes para o design responsivo de imagens e vídeos.

1) object-fit: fill: a imagem será redimensionada para preencher as dimensões definidas (a imagem pode ficar distorcida). É o valor padrão.

Exemplo (img de 1920x1080 em um espaço de 500x500):



2) object-fit: contain: o aspect ratio da imagem é preservado, mas a imagem é redimensionada para preencher as dimensões definidas.

Exemplo (img de 1920x1080 em um espaço de 500x500):



3) object-fit: cover: o aspect ratio da imagem é preservado, mas a imagem é redimensionada para preencher as dimensões definidas, cortando certas partes da imagem para evitar a distorção ou diminuição da imagem.

Exemplo (img de 1920x1080 em um espaço de 500x500):



Agora, o Object-position é uma propriedade que é utilizada para definir a posição de imagens ou vídeos dentro de seu contêiner. E ainda, é utilizada para a propriedade "cover".

O object-position aceita valores númericos, mas é mais prático se utilizado com porcentagem.
