7 - Combinadores

Antes de entender os combinadores, é necessário entender o que são é o "parent" (pai), "child" (filho) e "sibling" (irmão):



Os combinadores fazem parte dos seletores, e eles definem alguma relação dentro do seletor. Há quatro tipos diferentes de seletores:

1) Descendant selector (seletor de descendentes): este combinador corresponde a todos os elementos que são descendentes de um elemento especificado.

Ou seja, este combinador selecionará todos os elementos especificados dentro de um elemento, classe ou id.
