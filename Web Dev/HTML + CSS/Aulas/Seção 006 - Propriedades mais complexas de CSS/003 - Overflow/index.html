6 - PROPRIEDADES MAIS COMPLEXAS DE CSS

3 - Overflow

A propriedade overflow (transbordar) é utilizada para controlar o que acontece com o conteúdo de um elemento quando ele é grande demais para caber em determinada área.

Ainda, o overflow pode definir se um conteúdo será cortado da tela caso "transborde" ou se será adicionado uma barra de scroll.

A propriedade overflow pode receber os seguintes valores:

1) visible: é o valor padrão de overflow para os elemento. Os elementos "transbordarão" para fora da área e não será adicionada nenhuma barra de scroll;

Exemplo:

Elemento com "overflow: visible"









2) hidden: os elementos não "transbordarão" para fora da área (tudo que estiver fora da área ficará invisível) e não será adicionada nenhuma barra de scroll;

Exemplo:

Elemento com "overflow: hidden"


3) scroll: os elementos não "transbordarão" para fora da área, porém será adicionada uma barra de scroll para mostrar todo o conteúdo dentro da área;

Exemplo:

Elemento com "overflow: scroll"


4) auto: os elementos não "transbordarão" para fora da área, porém, apenas quando necessário, será adicionada uma barra de scroll para mostrar todo o conteúdo dentro da área;

Exemplo:

Elemento com "overflow: auto"


Elemento com "overflow: auto"



Todos estes valores também podem ser usados para o overflow direções diferentes, isto é, é possível definir um valor de overflow na vertical e outro na horizontal.

• overflow-x + overflow-y: define o overflow separadamente na direção vertical e horizontal. Aceita todos os valores utilizados no "overflow".

Exemplo de elemento com "overflow-x: scroll" e "overflow-y: hidden;":

Elemento com "overflow-x: scroll" e "overflow-y: hidden;"


OBS¹: os elementos com o "overflow: visible", quando "transbordarem" para fora de sua área limite, eles não ocuparão de fato o espaço de fora da área, isto é, sua parte visual estará fora da área e ficará por cima de outros elementos, mesmo ela apenas ocupando "fisicamente" a área que está inserida.

OBS¹: os elementos com o valor scroll ou auto criam uma barra de scroll, porém, é válido destacar que, a área que eles ocupavam "aumenta de tamanho" para poder receber os elementos que estão "transbordando", portanto, caso um elemento esteja "transbordando" para fora de uma área com o fundo azul, quando for colocado o valor scroll ou auto, todo o conteúdo que estava transbordando para fora fica dentro da área, porém o fundo azul não muda de tamanho, e sim o que acontece é apenas o deslocamento do texto na frente de um fundo.
