3 - TAGS BÁSICAS DE HTML

7 - Formulários

Aqui são compreendidas tags utilizadas para a criação e envio de formulários na web.

1) <form>: é a tag padrão para criação de formulários.

2) <input>: é a tag para adicionar caixas de texto onde é possível escrever.

2.1) type é utilizado para definir o tipo do input, podendo ser “text”, “password“, “button” ou “submit”

2.1.1) type="text" o texto dentro da caixa ficará normal.

2.1.2) type="password" o texto dentro da caixa ficará todo em asterisco (*).

Exemplo:



2.1.3) type="button" transforma a caixa de texto em um botão que não envia um formulário para o site quando clicado. Esse tipo de input não faz nada por padrão, e são comumente usados para atuar com o JavaScript ou AJAX.

Exemplo:



2.1.4) type="submit" transforma a caixa de texto em um botão que envia um formulário para o site, é a função padrão dele (é o botão usado para salvar o login de um site, por exemplo). Isso é melhor compreendido quando se estuda Back-End.

Exemplo:



2.1.5) type="radio" cria uma região onde é possível marcar apenas uma opção. É importante definir o name para que os inputs saibam que eles fazem parte de uma mesma seção (fazendo com que não seja possível marcar duas bolinhas).
